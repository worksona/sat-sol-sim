📚 API Documentation

Complete programmatic control over the solar system simulation 

// Global API Access
const api = window.SolarSystemAPI; // or import SatSolSim from 'sat-sol-sim'

// All methods are chainable for fluid control
api.simulation.setSpeed(2.0).start()
   .camera.setPreset(3).enableFreeMode()
   .satellites.highlight('LEO_COMM')
   .visuals.hideUI().enableNegativeMode();

🎮 Simulation Control

  • start() Start the simulation
  • pause() Pause the simulation
  • reset() Reset to initial state
  • setSpeed(speed) Set simulation speed (0.1-3.0)
  • getState() Get current simulation state
api.simulation.setSpeed(1.5);
api.simulation.pause();
const state = api.simulation.getState();

📷 Camera Management

  • setPreset(preset) Set camera preset (0-7)
  • enableFreeMode() Enable free camera mode
  • disableFreeMode() Disable free camera mode
  • enableFollowEarth() Follow Earth's orbital motion
  • setPosition(x, y, z) Set camera position directly
api.camera.setPreset(3); // Solar system view
api.camera.enableFreeMode();
api.camera.enableFollowEarth();

🛰️ Satellite Operations

  • highlight(type) Highlight satellite constellation
  • clearHighlight() Clear all highlighting
  • setConfiguration(config) Update satellite counts
  • getSatellites(filter) Get satellite data with filtering
  • simulateFailure(count) Simulate satellite failures
api.satellites.highlight('LEO_COMM');
api.satellites.setConfiguration({
  LEO_COMM: 100, MEO_NAV: 50
});

🌍 Celestial Bodies

  • getSun() Get Sun data and position
  • getPlanets() Get all planet data
  • getEarth() Get Earth position and data
  • getMoon() Get Moon position and data
  • getPlanet(name) Get specific planet data
const earth = api.celestialBodies.getEarth();
const planets = api.celestialBodies.getPlanets();
const mars = api.celestialBodies.getPlanet('mars');

🎨 Visual Controls

  • showOrbits() / hideOrbits() Toggle orbital rings
  • showTrails() / hideTrails() Toggle satellite trails
  • showUI() / hideUI() Toggle interface panels
  • enableNegativeMode() Enable inverted colors
  • getVisualState() Get current visual settings
api.visuals.hideUI();
api.visuals.enableNegativeMode();
api.visuals.showTrails();

⚡ Events & Utilities

  • on(event, callback) Subscribe to events
  • exportConfiguration() Export current settings
  • importConfiguration(config) Import saved settings
  • takeScreenshot() Capture simulation image
  • getSystemInfo() Get performance metrics
api.events.on('satelliteFailure', (data) => {
  console.log('Failure:', data);
});
api.utils.takeScreenshot();

💡 Code Examples

🎬 Automated Camera Tour

// Create an automated tour of the solar system
const tour = async () => {
  const presets = [0, 1, 2, 3, 4, 5, 6, 7];
  const names = ['Sun View', 'Sun Close', 'Earth Close', 'Moon View', 
                'Orbital View', 'Solar System', 'Earth Polar', 'Random Satellite'];
  
  for (let i = 0; i < presets.length; i++) {
    console.log(`Tour Stop ${i + 1}: ${names[i]}`);
    api.camera.setPreset(presets[i]);
    await new Promise(resolve => setTimeout(resolve, 5000));
  }
};

tour();

📊 Satellite Constellation Analysis

// Analyze satellite coverage and highlight patterns
const analyzeConstellation = (type) => {
  // Highlight the constellation
  api.satellites.highlight(type);
  
  // Get satellite data
  const satellites = api.satellites.getSatellites({ type });
  
  // Show trails for better visualization
  api.visuals.showTrails();
  
  // Set optimal camera view
  api.camera.setPreset(4); // Orbital view
  
  console.log(`${type} Constellation:`, {
    count: satellites.length,
    operational: satellites.filter(s => s.status === 'operational').length
  });
};

analyzeConstellation('LEO_COMM');

🎨 Presentation Mode Setup

// Configure simulation for professional presentations
const presentationMode = () => {
  // Hide UI for clean view
  api.visuals.hideUI();
  
  // Set slower speed for better observation
  api.simulation.setSpeed(0.5);
  
  // Enable trails for visual appeal
  api.visuals.showTrails();
  
  // Set wide solar system view
  api.camera.setPreset(5);
  
  // Optional: Enable dramatic negative mode
  // api.visuals.enableNegativeMode();
  
  console.log('Presentation mode activated');
};

presentationMode();

📡 Mission Planning Simulation

// Simulate satellite deployment and failure scenarios
const missionPlan = async () => {
  // Reset to clean state
  api.simulation.reset();
  
  // Deploy initial constellation
  api.satellites.setConfiguration({
    LEO_COMM: 50,
    MEO_NAV: 24,
    GEO_COMM: 12
  });
  
  // Monitor constellation
  api.events.on('satelliteFailure', (data) => {
    console.log(`Mission Alert: ${data.type} satellite failed`);
  });
  
  // Simulate operational period
  setTimeout(() => {
    console.log('Simulating satellite failures...');
    api.satellites.simulateFailure(3);
  }, 10000);
};

missionPlan();
