🚀 Overview

The Solar System API provides comprehensive programmatic control over the 3D Solar System Satellite Simulation. This powerful API enables developers to create custom applications, automate simulations, build educational tools, and create immersive space experiences.

Key Features

  • 🎮 Chainable Methods - Fluent interface for easy scripting
  • 📷 Advanced Camera System - 8 preset positions plus free camera mode
  • 🛰️ Satellite Management - Control 5 different satellite types
  • 🎨 Visual Effects - Orbits, trails, negative mode, UI controls
  • 🔔 Event System - Real-time event notifications
  • 💾 Configuration Management - Save/load complete simulation states
🎮 Interactive Playground 📝 Basic Examples 📷 Camera Tour 📋 Swagger UI

⚡ Quick Start

Installation

<!-- Include after the main simulation --> <script src="sat-sol-sim.js"></script> <!-- Or use minified version --> <script src="sat-sol-sim.min.js"></script>

Basic Usage

// All methods are chainable for fluid programming SolarSystemAPI .simulation.setSpeed(2.0) .camera.setPreset(3) .visuals.showTrails() .satellites.highlight('LEO_COMM'); // Event-driven programming SolarSystemAPI.events.on('initialized', function(data) { console.log('🌟 Simulation ready!', data); // Start your custom logic here SolarSystemAPI.simulation.start(); });
💡 Pro Tip

The API is designed for method chaining. You can combine multiple operations in a single fluent call for cleaner, more readable code.

🎮 Simulation Control

Control the core simulation state, timing, and playback with SolarSystemAPI.simulation.

start()
Starts or resumes the simulation
Returns: this (chainable)
SolarSystemAPI.simulation.start();
pause()
Pauses the simulation
Returns: this (chainable)
setSpeed(speed)
Sets the simulation speed multiplier
Parameters: speed (number) - Speed multiplier between 0.1 and 3.0
Returns: this (chainable)
// Slow motion SolarSystemAPI.simulation.setSpeed(0.5); // Double speed SolarSystemAPI.simulation.setSpeed(2.0); // Maximum speed SolarSystemAPI.simulation.setSpeed(3.0);
getState()
Gets comprehensive simulation state information
Returns: Object with simulation state details
const state = SolarSystemAPI.simulation.getState(); console.log('Current state:', state); // Returns: { paused: false, speed: 1.0, earthOrbitAngle: 45.2, ... }
🎯 Example: Automated Simulation Control
// Create an automated simulation sequence function runAutomatedDemo() { SolarSystemAPI.simulation .setSpeed(0.5) // Start slow .start(); setTimeout(() => { SolarSystemAPI.simulation.setSpeed(2.0); // Speed up }, 5000); setTimeout(() => { SolarSystemAPI.simulation.pause(); // Pause for analysis }, 10000); } // Run the demo runAutomatedDemo();

📷 Camera System

Advanced camera control with presets, free movement, and tracking modes via SolarSystemAPI.camera.

Camera Presets

Index Name Description Use Case
0 Sun View Default view from Sun to Earth Overview of Earth-Moon system
1 Sun Close 75% closer to Earth from Sun Detailed Earth observation
2 Earth Close Close orbit around Earth Satellite constellation analysis
3 Moon View From Moon looking at Earth Lunar perspective
4 Orbital View Satellite constellation overview Communication network analysis
5 Solar System Wide view of entire solar system Cosmic perspective
6 Earth Polar View from Earth's north pole Polar orbit visualization
7 Random Satellite Random satellite perspective Satellite POV experience
setPreset(presetIndex)
Sets camera to a specific preset position
Parameters: presetIndex (number) - Preset index (0-7)
Returns: this (chainable)
// Cinematic camera tour SolarSystemAPI.camera.setPreset(5); // Solar system view setTimeout(() => SolarSystemAPI.camera.setPreset(2), 2000); // Earth close setTimeout(() => SolarSystemAPI.camera.setPreset(7), 4000); // Satellite view
enableFreeMode() / disableFreeMode()
Enables/disables free camera mode for manual control
Returns: this (chainable)
// Enable free camera movement SolarSystemAPI.camera.enableFreeMode(); // Position camera manually SolarSystemAPI.camera .setPosition(100, 50, 200) .lookAt(0, 0, 0);
🎬 Example: Automated Camera Tour
function createCameraTour() { const presets = [0, 1, 2, 3, 4, 5, 6, 7]; let currentPreset = 0; function nextPreset() { SolarSystemAPI.camera.setPreset(presets[currentPreset]); currentPreset = (currentPreset + 1) % presets.length; } // Change camera every 3 seconds setInterval(nextPreset, 3000); nextPreset(); // Start immediately } // Launch the tour SolarSystemAPI.events.on('initialized', createCameraTour);

🛰️ Satellite Management

Comprehensive satellite constellation control with SolarSystemAPI.satellites.

Satellite Types

Type Full Name Orbit Purpose
LEO_COMM Low Earth Orbit Communications 160-2000 km Internet, phone, data services
LEO_WEATHER Weather Satellites 700-800 km Weather monitoring, climate
MEO_NAV Medium Earth Orbit Navigation 20,000 km GPS, GLONASS, Galileo
GEO_COMM Geostationary Communications 35,786 km TV, radio, telecommunications
MILITARY Military/Reconnaissance Various Defense, surveillance, intelligence
getAll()
Gets detailed information about all satellites
Returns: Array of satellite objects
const satellites = SolarSystemAPI.satellites.getAll(); console.log(`Total satellites: ${satellites.length}`); satellites.forEach(sat => { console.log(`${sat.type}: ${sat.status} (Health: ${sat.health}%)`); });
highlight(type) / unhighlight()
Highlights satellites of a specific type or removes all highlighting
Parameters: type (string) - Satellite type to highlight
Returns: this (chainable)
// Highlight communication satellites SolarSystemAPI.satellites.highlight('LEO_COMM'); // Show GPS constellation SolarSystemAPI.satellites.highlight('MEO_NAV'); // Clear all highlighting SolarSystemAPI.satellites.unhighlight();
setConfiguration(config)
Sets satellite constellation configuration
Parameters: config (Object) - Configuration with satellite counts
Returns: this (chainable)
// Custom constellation setup SolarSystemAPI.satellites.setConfiguration({ LEO_COMM: 12, // 12 communication satellites LEO_WEATHER: 8, // 8 weather satellites MEO_NAV: 24, // 24 GPS satellites GEO_COMM: 6, // 6 geostationary satellites MILITARY: 3 // 3 military satellites });
🎯 Example: Satellite Constellation Analysis
function analyzeSatelliteNetwork() { const satellites = SolarSystemAPI.satellites.getAll(); const byType = {}; // Group satellites by type satellites.forEach(sat => { if (!byType[sat.type]) byType[sat.type] = []; byType[sat.type].push(sat); }); // Analysis Object.keys(byType).forEach(type => { const sats = byType[type]; const operational = sats.filter(s => s.status === 'operational'); const avgHealth = sats.reduce((sum, s) => sum + s.health, 0) / sats.length; console.log(`${type}: ${operational.length}/${sats.length} operational, ${avgHealth.toFixed(1)}% avg health`); // Highlight if health is low if (avgHealth < 80) { SolarSystemAPI.satellites.highlight(type); } }); } // Run analysis every 10 seconds setInterval(analyzeSatelliteNetwork, 10000);

🎨 Visual Effects

Control visual elements and special effects with SolarSystemAPI.visuals.

showOrbits() / hideOrbits()
Shows/hides orbital rings for better spatial understanding
Returns: this (chainable)
showTrails() / hideTrails()
Shows/hides satellite movement trails
Returns: this (chainable)
enableNegativeMode() / disableNegativeMode()
Enables/disables negative lighting mode (inverted colors)
Returns: this (chainable)
🌟 Example: Dynamic Visual Effects
// Create a visual effects sequence function createVisualShow() { // Phase 1: Clean view SolarSystemAPI.visuals .hideOrbits() .hideTrails() .disableNegativeMode(); setTimeout(() => { // Phase 2: Show orbital structure SolarSystemAPI.visuals.showOrbits(); }, 2000); setTimeout(() => { // Phase 3: Add movement trails SolarSystemAPI.visuals.showTrails(); }, 4000); setTimeout(() => { // Phase 4: Dramatic negative mode SolarSystemAPI.visuals.enableNegativeMode(); }, 6000); } // Combine with camera movement for cinematic effect function cinematicExperience() { SolarSystemAPI.camera.setPreset(5); // Start with solar system view createVisualShow(); setTimeout(() => { SolarSystemAPI.camera.setPreset(2); // Move to Earth close }, 8000); }

🔔 Event System

React to simulation events with SolarSystemAPI.events.

on(eventName, callback)
Subscribe to simulation events
Parameters: eventName (string), callback (function)
Returns: this (chainable)
⚡ Example: Event-Driven Application
// Listen for key simulation events SolarSystemAPI.events .on('initialized', function(data) { console.log('🚀 Simulation initialized', data); startCustomLogic(); }) .on('satellite_failed', function(satellite) { console.log('⚠️ Satellite failure:', satellite); handleSatelliteFailure(satellite); }) .on('configuration_changed', function(config) { console.log('🔧 Configuration updated', config); updateAnalytics(config); }); function startCustomLogic() { // Your application logic here SolarSystemAPI.simulation.start(); } function handleSatelliteFailure(satellite) { // Highlight failed satellite SolarSystemAPI.satellites.highlight(satellite.type); // Show alert alert(`Satellite ${satellite.id} has failed!`); }

🛠️ Utilities

Utility functions for configuration management and system info with SolarSystemAPI.utils.

exportConfiguration()
Export complete simulation configuration
Returns: Object with complete configuration
importConfiguration(config)
Import and apply configuration
Parameters: config (Object) - Configuration to apply
Returns: this (chainable)
takeScreenshot()
Take screenshot of current simulation
Returns: Base64 encoded PNG image data URL
💾 Example: Configuration Management
// Save current configuration function saveSimulationState() { const config = SolarSystemAPI.utils.exportConfiguration(); localStorage.setItem('simulation_config', JSON.stringify(config)); console.log('Configuration saved!'); } // Load saved configuration function loadSimulationState() { const savedConfig = localStorage.getItem('simulation_config'); if (savedConfig) { const config = JSON.parse(savedConfig); SolarSystemAPI.utils.importConfiguration(config); console.log('Configuration loaded!'); } } // Take and download screenshot function captureScreenshot() { const dataURL = SolarSystemAPI.utils.takeScreenshot(); if (dataURL) { const link = document.createElement('a'); link.download = 'solar-system-' + Date.now() + '.png'; link.href = dataURL; link.click(); } }

💡 Advanced Examples

🎓 Educational Presentation System
class EducationalPresentation { constructor() { this.slides = [ { title: "Solar System Overview", camera: 5, visuals: { orbits: true, trails: false }, duration: 10000 }, { title: "Earth and Moon", camera: 0, visuals: { orbits: true, trails: true }, duration: 8000 }, { title: "Satellite Constellations", camera: 2, satellites: 'LEO_COMM', visuals: { orbits: false, trails: true }, duration: 12000 } ]; this.currentSlide = 0; } start() { this.showSlide(0); } showSlide(index) { const slide = this.slides[index]; // Set camera SolarSystemAPI.camera.setPreset(slide.camera); // Configure visuals if (slide.visuals.orbits) { SolarSystemAPI.visuals.showOrbits(); } else { SolarSystemAPI.visuals.hideOrbits(); } if (slide.visuals.trails) { SolarSystemAPI.visuals.showTrails(); } else { SolarSystemAPI.visuals.hideTrails(); } // Highlight satellites if (slide.satellites) { SolarSystemAPI.satellites.highlight(slide.satellites); } console.log(`📺 Slide ${index + 1}: ${slide.title}`); // Auto-advance setTimeout(() => { this.nextSlide(); }, slide.duration); } nextSlide() { this.currentSlide = (this.currentSlide + 1) % this.slides.length; this.showSlide(this.currentSlide); } } // Usage const presentation = new EducationalPresentation(); SolarSystemAPI.events.on('initialized', () => presentation.start());
📊 Performance Monitor
class PerformanceMonitor { constructor() { this.metrics = { fps: [], satellites: [], timestamps: [] }; this.startTime = Date.now(); } start() { setInterval(() => this.collect(), 1000); } collect() { const now = Date.now(); const satellites = SolarSystemAPI.satellites.getAll(); const systemInfo = SolarSystemAPI.utils.getSystemInfo(); this.metrics.timestamps.push(now - this.startTime); this.metrics.satellites.push(satellites.length); // Calculate operational percentage const operational = satellites.filter(s => s.status === 'operational'); const operationalPercent = (operational.length / satellites.length * 100).toFixed(1); console.log(`📊 Performance Report: Satellites: ${satellites.length} (${operationalPercent}% operational) Runtime: ${((now - this.startTime) / 1000).toFixed(1)}s Camera: Preset ${systemInfo.camera.preset} `); // Alert if performance degrades if (operationalPercent < 90) { console.warn('⚠️ Satellite network degraded!'); SolarSystemAPI.satellites.highlight('failed'); } } getReport() { return { runtime: Date.now() - this.startTime, metrics: this.metrics, averageSatellites: this.metrics.satellites.reduce((a, b) => a + b, 0) / this.metrics.satellites.length }; } } // Usage const monitor = new PerformanceMonitor(); SolarSystemAPI.events.on('initialized', () => monitor.start());
