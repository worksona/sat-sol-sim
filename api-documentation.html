üöÄ Overview

The Solar System API provides comprehensive programmatic control over the 3D Solar System Satellite Simulation. This powerful API enables developers to create custom applications, automate simulations, build educational tools, and create immersive space experiences.

Key Features

  • üéÆ Chainable Methods - Fluent interface for easy scripting
  • üì∑ Advanced Camera System - 8 preset positions plus free camera mode

  • üîî Event System - Real-time event notifications

Basic Usage

üí° Pro Tip

The API is designed for method chaining. You can combine multiple operations in a single fluent call for cleaner, more readable code.

üéÆ Simulation Control

Control the core simulation state, timing, and playback with SolarSystemAPI.simulation.

start()
Starts or resumes the simulation
Returns: this (chainable)
SolarSystemAPI.simulation.start();
pause()
Pauses the simulation
Returns: this (chainable)
setSpeed(speed)
Sets the simulation speed multiplier
Parameters: speed (number) - Speed multiplier between 0.1 and 3.0
Returns: this (chainable)
// Slow motion SolarSystemAPI.simulation.setSpeed(0.5); // Double speed SolarSystemAPI.simulation.setSpeed(2.0); // Maximum speed SolarSystemAPI.simulation.setSpeed(3.0);
getState()
Gets comprehensive simulation state information
Returns: Object with simulation state details
const state = SolarSystemAPI.simulation.getState(); console.log('Current state:', state); // Returns: { paused: false, speed: 1.0, earthOrbitAngle: 45.2, ... }
üéØ Example: Automated Simulation Control
// Create an automated simulation sequence function runAutomatedDemo() { SolarSystemAPI.simulation .setSpeed(0.5) // Start slow .start(); setTimeout(() => { SolarSystemAPI.simulation.setSpeed(2.0); // Speed up }, 5000); setTimeout(() => { SolarSystemAPI.simulation.pause(); // Pause for analysis }, 10000); } // Run the demo runAutomatedDemo();

üì∑ Camera System

Advanced camera control with presets, free movement, and tracking modes via SolarSystemAPI.camera.

Camera Presets

Index Name Description Use Case
0 Sun View Default view from Sun to Earth Overview of Earth-Moon system
1 Sun Close 75% closer to Earth from Sun Detailed Earth observation
2 Earth Close Close orbit around Earth Satellite constellation analysis
3 Moon View From Moon looking at Earth Lunar perspective
4 Orbital View Satellite constellation overview Communication network analysis
5 Solar System Wide view of entire solar system Cosmic perspective
6 Earth Polar View from Earth's north pole Polar orbit visualization
7 Random Satellite Random satellite perspective Satellite POV experience
setPreset(presetIndex)
Sets camera to a specific preset position
Parameters: presetIndex (number) - Preset index (0-7)
Returns: this (chainable)
// Cinematic camera tour SolarSystemAPI.camera.setPreset(5); // Solar system view setTimeout(() => SolarSystemAPI.camera.setPreset(2), 2000); // Earth close setTimeout(() => SolarSystemAPI.camera.setPreset(7), 4000); // Satellite view
enableFreeMode() / disableFreeMode()
Enables/disables free camera mode for manual control
Returns: this (chainable)
// Enable free camera movement SolarSystemAPI.camera.enableFreeMode(); // Position camera manually SolarSystemAPI.camera .setPosition(100, 50, 200) .lookAt(0, 0, 0);
üé¨ Example: Automated Camera Tour
function createCameraTour() { const presets = [0, 1, 2, 3, 4, 5, 6, 7]; let currentPreset = 0; function nextPreset() { SolarSystemAPI.camera.setPreset(presets[currentPreset]); currentPreset = (currentPreset + 1) % presets.length; } // Change camera every 3 seconds setInterval(nextPreset, 3000); nextPreset(); // Start immediately } // Launch the tour SolarSystemAPI.events.on('initialized', createCameraTour);

Type Full Name Orbit Purpose

getAll()
Gets detailed information about all satellites
Returns: Array of satellite objects
const satellites = SolarSystemAPI.satellites.getAll(); console.log(`Total satellites: ${satellites.length}`); satellites.forEach(sat => { console.log(`${sat.type}: ${sat.status} (Health: ${sat.health}%)`); });
highlight(type) / unhighlight()
Highlights satellites of a specific type or removes all highlighting
Parameters: type (string) - Satellite type to highlight
Returns: this (chainable)
// Highlight communication satellites SolarSystemAPI.satellites.highlight('LEO_COMM'); // Show GPS constellation SolarSystemAPI.satellites.highlight('MEO_NAV'); // Clear all highlighting SolarSystemAPI.satellites.unhighlight();
setConfiguration(config)
Sets satellite constellation configuration
Parameters: config (Object) - Configuration with satellite counts
Returns: this (chainable)
// Custom constellation setup SolarSystemAPI.satellites.setConfiguration({ LEO_COMM: 12, // 12 communication satellites LEO_WEATHER: 8, // 8 weather satellites MEO_NAV: 24, // 24 GPS satellites GEO_COMM: 6, // 6 geostationary satellites MILITARY: 3 // 3 military satellites });
üéØ Example: Satellite Constellation Analysis
function analyzeSatelliteNetwork() { const satellites = SolarSystemAPI.satellites.getAll(); const byType = {}; // Group satellites by type satellites.forEach(sat => { if (!byType[sat.type]) byType[sat.type] = []; byType[sat.type].push(sat); }); // Analysis Object.keys(byType).forEach(type => { const sats = byType[type]; const operational = sats.filter(s => s.status === 'operational'); const avgHealth = sats.reduce((sum, s) => sum + s.health, 0) / sats.length; console.log(`${type}: ${operational.length}/${sats.length} operational, ${avgHealth.toFixed(1)}% avg health`); // Highlight if health is low if (avgHealth < 80) { SolarSystemAPI.satellites.highlight(type); } }); } // Run analysis every 10 seconds setInterval(analyzeSatelliteNetwork, 10000);

üîî Event System

React to simulation events with SolarSystemAPI.events.

on(eventName, callback)
Subscribe to simulation events
Parameters: eventName (string), callback (function)
Returns: this (chainable)
‚ö° Example: Event-Driven Application
// Listen for key simulation events SolarSystemAPI.events .on('initialized', function(data) { console.log('üöÄ Simulation initialized', data); startCustomLogic(); }) .on('satellite_failed', function(satellite) { console.log('‚ö†Ô∏è Satellite failure:', satellite); handleSatelliteFailure(satellite); }) .on('configuration_changed', function(config) { console.log('üîß Configuration updated', config); updateAnalytics(config); }); function startCustomLogic() { // Your application logic here SolarSystemAPI.simulation.start(); } function handleSatelliteFailure(satellite) { // Highlight failed satellite SolarSystemAPI.satellites.highlight(satellite.type); // Show alert alert(`Satellite ${satellite.id} has failed!`); }

üéì Educational Presentation System
class EducationalPresentation { constructor() { this.slides = [ { title: "Solar System Overview", camera: 5, visuals: { orbits: true, trails: false }, duration: 10000 }, { title: "Earth and Moon", camera: 0, visuals: { orbits: true, trails: true }, duration: 8000 }, { title: "Satellite Constellations", camera: 2, satellites: 'LEO_COMM', visuals: { orbits: false, trails: true }, duration: 12000 } ]; this.currentSlide = 0; } start() { this.showSlide(0); } showSlide(index) { const slide = this.slides[index]; // Set camera SolarSystemAPI.camera.setPreset(slide.camera); // Configure visuals if (slide.visuals.orbits) { SolarSystemAPI.visuals.showOrbits(); } else { SolarSystemAPI.visuals.hideOrbits(); } if (slide.visuals.trails) { SolarSystemAPI.visuals.showTrails(); } else { SolarSystemAPI.visuals.hideTrails(); } // Highlight satellites if (slide.satellites) { SolarSystemAPI.satellites.highlight(slide.satellites); } console.log(`üì∫ Slide ${index + 1}: ${slide.title}`); // Auto-advance setTimeout(() => { this.nextSlide(); }, slide.duration); } nextSlide() { this.currentSlide = (this.currentSlide + 1) % this.slides.length; this.showSlide(this.currentSlide); } } // Usage const presentation = new EducationalPresentation(); SolarSystemAPI.events.on('initialized', () => presentation.start());
üìä Performance Monitor
class PerformanceMonitor { constructor() { this.metrics = { fps: [], satellites: [], timestamps: [] }; this.startTime = Date.now(); } start() { setInterval(() => this.collect(), 1000); } collect() { const now = Date.now(); const satellites = SolarSystemAPI.satellites.getAll(); const systemInfo = SolarSystemAPI.utils.getSystemInfo(); this.metrics.timestamps.push(now - this.startTime); this.metrics.satellites.push(satellites.length); // Calculate operational percentage const operational = satellites.filter(s => s.status === 'operational'); const operationalPercent = (operational.length / satellites.length * 100).toFixed(1); console.log(`üìä Performance Report: Satellites: ${satellites.length} (${operationalPercent}% operational) Runtime: ${((now - this.startTime) / 1000).toFixed(1)}s Camera: Preset ${systemInfo.camera.preset} `); // Alert if performance degrades if (operationalPercent < 90) { console.warn('‚ö†Ô∏è Satellite network degraded!'); SolarSystemAPI.satellites.highlight('failed'); } } getReport() { return { runtime: Date.now() - this.startTime, metrics: this.metrics, averageSatellites: this.metrics.satellites.reduce((a, b) => a + b, 0) / this.metrics.satellites.length }; } } // Usage const monitor = new PerformanceMonitor(); SolarSystemAPI.events.on('initialized', () => monitor.start());
